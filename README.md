# issues

issues is a demonstration of a minimal issue-tracking application using a flask webapp, a flask-based api, and sqlalchemy models with a sqlite db.

## installation

```bash
pipenv install
```

## getting started

enter the virtual environment

```bash
pipenv shell
```

start the services:

```bash
python3 api.py
python3 site.py
```
